# yii_basic_template
Yii basic template

### Quick start

```bash
composer install
cp .env.dev .env
yii serve --docroot public
```
Visit http://localhost:8080/

### Run console  command

```bash
yii console/test/env
yii console/test/runtask
```

### Migration

```bash
yii migrate
yii migrate/down 1
```
